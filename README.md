# Memory Format library

Formatter class to get the value of memory as string.

## Data types
 - 1 bit
 - 8 bit
 - 16 bit
 - 32 bit
 - 64 bit

## Formats
 - Binary
 - Signed decimal
 - Unsigned decimal
 - Hexadecimal
 - Octal
 - Float

## Endianness
 - Host
 - Big
 - Little

In addition, the swapping of 16 and 32 bit blocks is supported
